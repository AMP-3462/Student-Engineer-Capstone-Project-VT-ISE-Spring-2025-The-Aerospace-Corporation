# Student-Engineer-Capstone-Project
The Aerospace Corporation wants to be able to view CAMEO files without having CAMEO open. This allows the organization to not have as many licenses purchased just for decision makers to access Systems Diagrams.

* Virginia Tech Industrial and Systems Engineering Spring 2025 Student Engineer Capstone Project. Team: Achyut Patel, Alex Harfouche, Derek Spratley.
